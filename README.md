# WARP+ Keys Collector

A simple tool that automatically collects 15 WARP+ Keys once every hour.<br>
နာရီတိုင်း warpplus သော့ 15 ခု အလိုအလျောက်စုဆောင်းပေးသည့် ရိုးရှင်းသောကိရိယာတစ်ခု.
